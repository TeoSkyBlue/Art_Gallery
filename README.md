# Art_Gallery
 Art Gallery website made by two undergrads as part of a university assignment, used to be deployed on heroku under
 the link: https://artgallery34.herokuapp.com/collection

-ExpressJS and a MongoDB cluster were used for the back-end.  
-Bootstrap and custom CSS/Javascript(ES6) were used for its front-end.  
-Simplistic yet mobile friendly design.
